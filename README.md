# chatbot-YM-test
Test Chatbot for pizza Delivery(Without nltk)

## Pre-requisites
 * Python 3x

## How to run app
 * run command ```pip install -r requirements.txt```
 * run command ```python app.py```

## App Architecture
 * Python Backend
 * HTML, CSS and JavaScript Frontend
 * SQLite3 Database
 
## How to use
 * On page, Click on order
 * Click on any size
